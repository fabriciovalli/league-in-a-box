02 How this differs from what's already out there

Existing tools are built for the club. This is built for whoever runs the league.

ClubZap / Pitchero
Built for the club, not the league organiser
Captains need an account to submit a score
Payments and membership come first, table second
LeagueRepublic
Reviewers call it overwhelming — too many settings
Desktop-era interface, not mobile-first
Generic look, not shaped for Irish grassroots leagues
League in a box
Built for the league organiser first
Captain submits from a link, no account needed
The table is the whole product, not a buried tab

League subscriptions are the distribution channel, not the revenue plan — every league page is seen weekly by every club playing in it. The club tier, sold per club rather than per league, is where this becomes a real business. Penetration within a county matters more than price: going county board by county board (the GAA's own structure) turns one sale into dozens of warm introductions.
